本项目是自己学习flutter的一个app,ios因为没有帐号也没有环境所以就...

功能比较简单，流程：登录->图书展示->增加购物车->选择借阅的书->选择借阅时间->下单->借阅记录

主要涉及技术：
1、app：以bloc进行交互
2：api：golang
3：db：mysql
